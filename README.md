# ♻️ EcoScan — Save Environment with Plastic Recycle

A web platform that allows users to **scan or photograph a plastic bottle** to identify its plastic type and check whether it is recyclable or not — promoting eco-friendly waste management.

---

## 🌍 About the Project

EcoScan is a web-based application designed to make plastic recycling easier and more accessible for everyone. Users can simply take a photo or scan a plastic bottle, and the platform will:

- 🔍 Identify the **type of plastic**
- ♻️ Check if it is **recyclable or not**
- 💡 Provide **eco-friendly disposal guidance**
- 🏆 Reward users with **points** for recycling efforts

---

## ✨ Features

- 📸 **Plastic Scanner** — Scan or photograph any plastic bottle to check recyclability
- ♻️ **Recyclability Check** — Instant results on whether the plastic can be recycled
- 🏆 **Rewards Program** — Earn points for every recyclable item scanned
- 🔐 **User Authentication** — Secure login and registration system
- 📱 **Responsive Design** — Works on both desktop and mobile devices
